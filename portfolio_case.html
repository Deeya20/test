About

At a Glance:

I am beginner to UI/UX design(had no real project) and chose an e-commerce project with Chatgpt’s help. I worked for a week on this project named Virtushop an app selling three main product categories(electronics & gadgets, Home appliance and Fashion). While I did not finish the entire design, I completed 50%.

Project Objective:

The primary Objective of Virtushop was to design an ecommerce app that makes user experience easy by providing distinct pages for each of its three main categories. This will simplify the overall user journey, promoting a straightforward and efficient shopping experience.

User Research:

Since I did not had a team or client to chat with, I took a different route for Virtushop’s research like followings:

  • Collaborating with ChatGpt:

    • I talked to ChatGpt to get ideas and simulate user interaction. I know it wont be like real interview, but it helped me with various things like what e-commerce app should have or how it should work.

  • Inspired By Other App:

    • I looked at various app like amazon and daraz to see how successful one works. For this project I did install daraz app and made account to understand how each element works like after entering app what home page looks, what happens when product is clicked and many more. The way of learning might be different, but its only way of making Virtushop the best I can do.

    Usability Test:

    As there was no real project and client, I conducted a usability test with my 3 friends who are learning frontend development(because they will have some knowledge about Figma and UI). They used prototype I shared through Figma. Based on their feedbacks, I made some improvement:

  • Using other secondary color for text instead using only black and white for better readability.
  • Highlighting categories to make it easier for user to navigate.
  • Adding popup message after user click add to cart button for more interactive experience.
